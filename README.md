# 🌊 ADT_MA_Python

Projeto em **Python** para processamento e análise de dados de **ADT (Absolute Dynamic Topography)**, com foco em aplicações oceanográficas e estudo do nível do mar.

Este repositório reúne scripts e rotinas para leitura, manipulação, extração e análise de dados (ex: arquivos `.mat`, NetCDF ou similares), permitindo avaliar variações espaciais e temporais da topografia dinâmica absoluta.

---

## 📌 Objetivos do Projeto

- Processar dados de **ADT**
- Realizar análises temporais e espaciais
- Extrair séries temporais em pontos específicos
- Gerar mapas e gráficos para estudos oceanográficos
- Automatizar rotinas de tratamento de dados

---

## 🗂 Estrutura do Repositório

O repositório contém principalmente:

- Scripts em Python (`.py`)
- Arquivos auxiliares em formato `.mat`
- Rotinas de análise e visualização

---

## ⚙️ Requisitos

Recomenda-se utilizar Python **3.8+**.

Bibliotecas comuns utilizadas:

- `numpy`
- `scipy`
- `matplotlib`
- `pandas`
- `xarray`
- `netCDF4`
